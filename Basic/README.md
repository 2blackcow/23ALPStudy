# Basic 

## 목표
- 문제풀이를 위한 기초적인 자료구조 맟 알고리즘 공부
- 문제풀이에 대한 방법을 알기
- 기업 코딩 테스트를 대비

## 형식
- 주 1회, 대면으로 진행
- 멘토링이 아닌 전체 참여 방식
  
## 선수 조건
- 기초적인 프로그래밍 작성법에 대해 알아야합니다
  - 기준 : [ALPS 2023 Spring Basic 선수 조건](https://www.acmicpc.net/workbook/view/13740) 해결
